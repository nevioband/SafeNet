# SafeNet Security

SafeNet Security ist eine deutschsprachige Lernplattform im Web. 

## Was bietet die Plattform?
- Passwort-Analyse
- Passwort-Generator
- Tresor für Passwörter und Notizen
- Infos zu Phishing, Bruteforce, Social Engineering, Keyloggern und Wörterbuchangriffen
- Tutorials und News

Alle sensiblen Daten werden sicher verschlüsselt gespeichert. Die Nutzung erfolgt komplett im Browser.
