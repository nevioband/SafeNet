# SafeNet Security

[Deutsches README](README.md)

SafeNet Security is a German-language web platform for learning about password security and digital attack methods. The site offers various tools and information to help users understand and improve their digital safety.

## Features
- Password analysis
- Password generator
- Vault for passwords and notes
- Information about phishing, brute force, social engineering, keyloggers, and dictionary attacks
- Tutorials and news

All sensitive data is securely encrypted. The platform runs entirely in the browser.
