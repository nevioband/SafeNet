🚀 Installation in 3 Schritten

1
Installer herunterladen – Klicke oben auf «Installer herunterladen» und speichere die Datei.
2
Setup starten – Doppelklick auf SafeNet.Security.Setup.1.0.0.exe. Falls Windows SmartScreen erscheint: «Weitere Informationen» → «Trotzdem ausführen».
3
App öffnen – Nach der Installation findest du SafeNet Security auf deinem Desktop und im Start-Menü.
